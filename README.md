### OBS Setup

- `obs-setup/ah-filming-setup.json` - OBS scene collection for the filming setup module. 4 horizontal scenes: Talking to Camera, Camera + Task, Camera + iPad, Camera + iPad Overlay.

## How to download

Click the file in GitHub, then click **Download raw file** (or right-click the "Raw" button > Save Link As).
